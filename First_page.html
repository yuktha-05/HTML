HTML is the standard markup language, it describes the structure of a web page.An HTML element is defined by a start tag,some content,and an end tag.In this code :-


1.)!DOCTYPE html--> this declares that the document is HTML5 document.
2.)html-->this is the root element.
3.)head-->this contains meta information of HTML page.
4.)title-->Specifies a title.
5.)body-->defines the document and only the content inside the body will be displayed.
6.)h1-->large heading.
7.)h3-->bit lesser large than h1 heading.
8.)p-->this defines a paragraph.
